Questions

  1. Name a real-life situation where you would need to use an `if` statement.
  2. I would use an if statement when a website asks me to enable cookies. For example: if a website requires cookies, I might choose to decline or accept based on my preference.
  3. Explain the difference between `==` and `===`. Why is strict equality usually better?
  4. `===` (strict equality) is going to compare two values and their data types. If these two values are of different data types, false will be returned. `==` will compare the values themselves, converting differing data types into a type that can be compared.
  5. Give an example (in your own words) of when you would use `&&` (AND) vs `||` (OR).
  6. You would use `&&` and `||` to extend specific conditions for if statements.
  7. What does the `!` (NOT) operator do? Give an example of a situation you would use it.
  8. The NOT operator says that if a specific condition isn't what is expected, then do something. An example of this would be if favorite color != red, then end.
